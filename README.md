# Flight-Scheduling-and-Fuel-Optimization-System
Graph-based flight scheduling and fuel optimization system implementing Dijkstra’s algorithm with time constraints, integrated with CSV data handling and a visualization dashboard.
## Overview
This project implements a flight scheduling and fuel optimization system using graph algorithms. It models flight routes as a network and computes optimal paths based on fuel cost while handling scheduling constraints.

The project combines algorithmic design with practical data handling and visualization to simulate real-world flight planning scenarios.
## Key Highlights
- Designed a system using graph-based modeling of flight networks  
- Implemented Dijkstra’s algorithm with constraints for route optimization  
- Integrated CSV-based data handling for scalability  
- Built an interactive dashboard for visualization and analysis
## Problem Statement
Airline route planning involves optimizing flight paths while considering constraints such as fuel cost and scheduling conflicts. This project aims to simulate this problem using algorithmic approaches and analyze optimal routing strategies.
